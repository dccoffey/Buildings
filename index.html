Key Features

  • Fully interactive: Toggle SF/MF, coarse/fine detail, discounted/undiscounted, and adjustable discount rate (1-7%)
  • Traceable: Every number is clickable and links to a provenance table with source, formula, and bibliography
  • Self-contained: No external network calls; all JS/CSS bundled locally for offline use and GitHub Pages deployment
  • Transparent: Methods & Assumptions fully documented; friction score formula is explicit and interpretable
  • Research-grounded: Sources include U.S. Census, EIA, NAHB, NMHC, NIST, CII, McKinsey, and peer-reviewed literature

How to Use

  1. Start with the Sankey diagram to understand cost magnitude and proportions
  2. Use the Friction Map to identify where process friction adds cost and delay
  3. Click any number to see its source and calculation in the provenance table
  4. Use the Methods page to understand all assumptions and limitations
